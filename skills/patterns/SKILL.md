# JavaScript Patterns Skill

## Overview
Master JavaScript design patterns and architectural approaches for maintainable, scalable applications.

## Topics Covered

### Creational Patterns
- Factory pattern
- Singleton pattern
- Builder pattern
- Prototype pattern
- Module pattern

### Structural Patterns
- Decorator pattern
- Facade pattern
- Adapter pattern
- Proxy pattern
- Composite pattern

### Behavioral Patterns
- Observer pattern
- Strategy pattern
- Command pattern
- State pattern
- Mediator pattern

### Modern Patterns
- Pub/Sub architecture
- Middleware pattern
- Higher-order functions
- Composition over inheritance
- Dependency injection

### Architectural Patterns
- MVC/MVP/MVVM
- Flux/Redux patterns
- Event-driven architecture
- Clean architecture in JS
- Micro-frontend patterns

## Prerequisites
- JavaScript fundamentals
- ES6+ features
- Object-oriented concepts

## Learning Outcomes
- Apply appropriate patterns
- Write maintainable code
- Structure large applications
- Refactor legacy code
