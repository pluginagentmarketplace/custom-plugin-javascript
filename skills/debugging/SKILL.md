# JavaScript Debugging Skill

## Overview
Master debugging JavaScript applications using browser DevTools, Node.js debugger, and advanced troubleshooting techniques.

## Topics Covered

### Chrome DevTools
- Sources panel navigation
- Breakpoint types (line, conditional, DOM)
- Call stack analysis
- Watch expressions
- Live editing

### Node.js Debugging
- --inspect flag usage
- VS Code debugging
- ndb debugger
- Remote debugging
- Async stack traces

### Memory Debugging
- Heap snapshots
- Memory profiler
- Allocation timeline
- Memory leak detection
- Garbage collection analysis

### Performance Debugging
- Performance panel
- Flame charts
- Long task identification
- Network waterfall
- Lighthouse audits

### Common Issues
- Async debugging patterns
- Promise debugging
- Event listener debugging
- Source map configuration

## Prerequisites
- JavaScript fundamentals
- Browser basics
- Node.js understanding

## Learning Outcomes
- Debug effectively with DevTools
- Find and fix memory leaks
- Identify performance bottlenecks
- Debug async code efficiently
