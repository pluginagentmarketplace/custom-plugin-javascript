- Debouncing and throttling

- Code splitting
- Dynamic imports

- Dead code elimination

- Closure memory patterns
- WeakMap and WeakSet

### Web Vitals
- LCP optimization
- FID/INP improvement
- CLS reduction
- TTFB optimization
- Performance budgets

### Profiling Tools

- Lighthouse CI
- Web Vitals library
- Bundle analyzers
- Real user monitoring

- Browser APIs
- Build tool basics

## Learning Outcomes
- Profile and optimize runtime
- Reduce bundle sizes
- Improve Core Web Vitals
- Set up performance monitoring
