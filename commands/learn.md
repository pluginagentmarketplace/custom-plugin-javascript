# /learn - Start Your JavaScript Journey

Welcome to the comprehensive JavaScript learning plugin! This command helps you select your learning path and get started with structured, agent-guided learning.

## How to Use

Type `/learn` to get started with your personalized learning journey.

## Learning Paths

### Path 1: Complete Beginner (Recommended)
Start from the very beginning and build your JavaScript foundation systematically.

**Duration:** 4-6 weeks (10-15 hours/week)

1. **Week 1-2:** JavaScript Fundamentals Agent
   - Variables, data types, operators
   - Control flow statements
   - Basic debugging
   - Use `fundamentals` skill for practice

2. **Week 3:** Functions & Scope Agent
   - Function declarations and expressions
   - Understanding scope
   - Closures and advanced patterns
   - Use `functions` skill for deep learning

3. **Week 4:** Objects & Arrays Agent
   - Object creation and manipulation
   - Array methods and transformations
   - Prototypal inheritance basics
   - Use `data-structures` skill

4. **Week 5:** Asynchronous JavaScript Agent
   - Callbacks to promises
   - async/await syntax
   - Error handling
   - Use `asynchronous` skill

5. **Week 6:** DOM & Browser APIs Agent
   - DOM selection and manipulation
   - Event handling
   - Fetch API and storage
   - Use `dom-apis` skill

### Path 2: Intermediate (Have basics)
Skip to advanced topics if you know fundamentals.

**Duration:** 2-3 weeks

1. Modern ES6+ Agent
   - Classes, arrow functions
   - Destructuring and spread
   - Modules and imports
   - Use `modern-javascript` skill

2. JavaScript Ecosystem Agent
   - npm and package management
   - Build tools
   - Testing frameworks
   - Use `ecosystem` skill

### Path 3: Quick Refresh (Just need to review)
**Duration:** 1 week

- Focus on specific agents based on your weak areas
- Use skills for quick reference
- Take the `/assess` command to evaluate your level

## Tips for Success

- **Set a schedule** - Dedicate consistent time (e.g., 1-2 hours daily)
- **Practice code** - Run all code examples yourself
- **Do the projects** - Apply learning in real projects
- **Take notes** - Document key concepts
- **Use /assess** - Evaluate your progress
- **Ask questions** - I'm here to help clarify anything

## Next Steps After Learning

After completing your chosen path:

1. **Build Projects** - Use `/projects` command
2. **Specialize** - Deep dive into React, Node.js, etc.
3. **Contribute** - Join open-source projects
4. **Stay Updated** - Keep learning with new frameworks
