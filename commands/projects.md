## Project Resources

### APIs for Projects
- **Weather:** OpenWeatherMap, WeatherAPI
- **Movies:** OMDB, TMDB
- **Random Data:** JSONPlaceholder, Faker
- **News:** NewsAPI
- **Quotes:** QuotesAPI

- **Markdown:** marked.js, markdown-it
- **Charts:** Chart.js, D3.js
- **Dates:** date-fns, dayjs
- **HTTP:** axios (after learning fetch)
- **UI:** Bootstrap, Tailwind (after JS mastery)

### Tools for Projects
- **Code:** VS Code, CodePen, Replit
- **Design:** Figma, Adobe XD
- **Hosting:** Netlify, Vercel, GitHub Pages

## Tips for Success

1. **Start Small** - Begin with beginner projects
2. **Follow the Agents** - Use agents for help while building
3. **Use Skills for Reference** - Check `/skills` for specifics
4. **Build in Steps** - Don't try to build everything at once
5. **Debug Systematically** - Use console.log and debugger
6. **Read Others' Code** - See how professionals solve problems
7. **Refactor Often** - Improve code as you learn
8. **Deploy Early** - Get your projects live
9. **Share Your Work** - Build your portfolio
